# How to Run This Project

Simple steps for beginners.

---

## Backend (infiApBackend)

Open terminal and type:

```bash
cd infiApBackend
npm install
npm run dev
```

Server starts at `http://localhost:3000`.

---

## Desktop Web (infiApHRMS_Desktop)

Open a **new** terminal and type:

```bash
cd infiApHRMS_Desktop
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

---

## Mobile App (infiApHRMS_Mobile)

Open a **new** terminal and type:

```bash
cd infiApHRMS_Mobile
npm install
npx expo start
```

Then:

- Press `a` for Android emulator
- Press `i` for iOS simulator (Mac only)
- Or scan the QR code with the **Expo Go** app on your phone

---

## Quick Tip

Run all three together by opening **three separate terminals**.

Start the **Backend first**, then **Desktop**, then **Mobile**.
